# NatureLife

![Projeto NatureLife no Figma](./img/projeto-figma.png)

Atividade desenvolvida no **SENAI Jandira** em dupla.  
O objetivo foi criar uma **landing page** entre as opções dadas pelo professor.  
Escolhemos o tema **Natureza** por ser simples e por nos permitir aprender diferentes tipos de formatação utilizando apenas **HTML** e **CSS**.

## Tecnologias Utilizadas
- HTML5  
- CSS3  
- Google Fonts  

## Estrutura do Projeto
- `index.html` → página principal  
- `css/reset.css` → redefinição dos estilos padrão  
- `css/style.css` → estilos principais da página  

## Descrição
A página apresenta um layout moderno com:
- Cabeçalho com menu e botão de login;  
- Texto principal com título, descrição e botão “Book Now”;  
- Rodapé com links para redes sociais;  
- Fundo com imagem e efeito de vidro (*blur*).
